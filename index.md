Privacy Policy for Dev
Last updated: 2026-01-31
Package name: com.dev.app
This Privacy Policy describes how the Dev app ("the App") collects, uses,
and shares information when you use the App.
Information We Collect
Chat content: Messages you type and the responses generated by the App.
Device data: Basic device information required by the operating system.
Usage data: Message count and timestamps for daily limits.
Anonymous user ID (optional): If Firebase is enabled, the App creates an
anonymous UID to track usage limits.
API key (optional): If you enter an OpenAI API key, it is stored locally
on your device. It is not uploaded by the App.
How We Use Information
Provide and improve the chat experience.
Enforce usage limits and prevent abuse.
Troubleshoot errors and improve stability.
Data Storage
Chat history is stored locally on your device.
If Firebase is enabled, usage counts and anonymous UID are stored in
Firebase services.
Sharing
OpenAI: Your message content is sent to the OpenAI API to generate replies.
Firebase (optional): Anonymous usage data may be sent to Firebase for
usage tracking if enabled.
Your Choices
You can delete local chat history in Settings.
You can remove your locally stored API key in Settings.
Security
We take reasonable steps to protect data, but no system is 100% secure.
Children
The App is not intended for children under 13. If you believe a child has
provided personal data, contact us to remove it.
